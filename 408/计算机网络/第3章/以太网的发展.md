# 吉比特以太网
吉比特以太网支持半双工方式吗? :: 支持,10以上吉比特以太网就不支持 ^1662648890992

千兆以太网为了保证最小帧长仍为64B,可以使用==载波延伸==的方法来实现,即当MAC帧长度不足以512字节,在MAC帧尾部添加一些特殊的字符,使其增大到512字节
^1662648890994

千兆以太网支持分组==突发==功能,即多个短帧连续发送,可以将第一个短帧用载波延伸的方法填充,而后面的一系列短帧不用填充就可以一个接一个发送,它们之间只需空开必要的帧间最小间隔即可.
^1662648890997
