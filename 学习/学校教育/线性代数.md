# 线性代数中的线性方程组
线程方程组相容的条件：没有形如\[0 ... 0 b\] b!=0的形式
- 若线性方程组相容：
	- 没有自由变量时，唯一解
	- 有自由变量时，无穷多解
- 不相容时，自由变量没有作用
⚠️方程组有自由变量不能保证方程组有解。
